# BLOWTORCH

AI-powered dating platform

## Tech Stack
- **Frontend**: React + TypeScript + Vite
- **Backend**: Node.js + Express + PostgreSQL
- **AI**: Python + FastAPI (scikit-learn)
- **DevOps**: Docker
