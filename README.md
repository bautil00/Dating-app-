# BLOWTORCH

AI-powered dating platform

## Tech Stack
- **Frontend**: React + TypeScript + Vite
- **Backend**: Python + FastAPI + PostgreSQL
- **AI**: Python (integrated into backend via API calls)
- **DevOps**: Docker
